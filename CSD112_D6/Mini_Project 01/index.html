CSS Box Model

All HTML elements can be considered as the boxes in CSS. The term "Box Model" is used when we talk about design and layout. It is very much used to apply the different CSS properties such as margins, padding, borders etc. to the block elements.

The CSS box model is essentially a box that wraps around every HTML element. If you see the CSS box model after inspecting the page, it consists of: margins, borders, padding and then the inner actual content.
